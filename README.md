# Шарапов Даниил. Написать программу - Хомячок ищет путь в лабиринте.

Лабиринт задается визуальными символами.
<pre>
S..xx..xxx
...xx..xx.
...x..x.x.
xx...xx..x
xxx.......
......xxx.
......xxxF
</pre>
где
S - старт
. - свободное пространство
F - финиш
x - препятствие

Можно ходить только по горизонтали и вертикали.
Надо вывести подобную картинку, где будет обозначен путь хомячка при помощи Звездочек.
<pre>
S..xx..xxx
*..xx..xx.
***x..x.x.
xx**.xx..x
xxx*******
......xxx*
......xxxF
</pre>

* Код писать на C++17.
* Без использования сторонних библиотек. Можно использовать стандартную библиотеку C и STL.
* Программа работает из командной строки.
* Без использования обработки исключений.
* Программа считывает лабиринт из файла input.txt. Результат выводит в stdout.
* Оцениваться будет как эффективность работы, так и качество кода. Не забывайте писать комментарии.
